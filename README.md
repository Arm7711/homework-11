# Restaurant API

## Project Description
FoodSpot API is a backend service for managing restaurants, products, and users. It supports image uploads and handles file storage efficiently.

Features:
- Single profile picture per user
- Single cover image per restaurant
- Multiple images for products (up to 5)
- Automatic cleanup of replaced or deleted files
- Validation of file types and sizes
- Organized folder structure for storage
- Relative file paths saved in the database
